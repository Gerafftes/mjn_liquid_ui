# Security Policy

## Supported versions

Security fixes are provided for the latest version published on
[pub.dev](https://pub.dev/packages/mjn_liquid_ui). Older releases may not
receive backported fixes.

| Version | Supported |
| --- | --- |
| Latest pub.dev release | Yes |
| Older releases | Not guaranteed |

## Reporting a vulnerability

Do not disclose suspected vulnerabilities in a public issue, discussion, or
pull request.

Use GitHub's private vulnerability reporting flow from the repository's
[Security page](https://github.com/Gerafftes/mjn_liquid_ui/security). If that
flow is unavailable, contact [@Gerafftes](https://github.com/Gerafftes) through
the private contact options on the profile and request a private reporting
channel.

Please include:

- The affected package version and platform.
- A clear description of the impact.
- Reproduction steps or a minimal proof of concept.
- Any known mitigations or workarounds.
- Whether the issue has been disclosed elsewhere.

You should receive an acknowledgement within seven days. After validation, the
maintainer will coordinate a fix and disclosure timeline based on severity.
Please allow reasonable time for a release before publishing details.

General bugs without a security impact should use the public bug report form.
